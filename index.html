Design For AIR 시작 가이드

Design For AIR는 nn.design 배포판의 데스크톱 앱입니다. 이 페이지는 처음 설치하는 사용자와 팀 내 가이드를 만드는 개발자를 위해 설치, 열기, 리믹스, 설정 흐름을 한곳에 정리합니다.

최신 설치 파일 전체 릴리스

1. 파일 선택

macOS 사용자는 최신 릴리스에서 .dmg 파일을 받습니다. 업데이트용 payload는 앱이 자동으로 처리하므로 직접 열 필요가 없습니다.

2. 앱 설치

DMG를 열고 Design For AIR를 Applications 폴더로 이동합니다. 기존 버전이 있다면 종료 후 교체합니다.

3. 첫 실행

보안 확인이 나오면 배포 출처를 확인한 뒤 실행합니다. 첫 실행 후 설정 화면에서 언어와 에이전트 상태를 점검합니다.

개발자 검증 기준: 앱 버전은 릴리스 태그보다 낮아야 업데이트가 감지됩니다. 같은 버전의 앱과 feed를 비교하면 최신 상태로 표시됩니다.
1

열기

작업할 프로젝트 폴더나 사내 템플릿 폴더를 엽니다. 처음에는 작은 화면 단위로 시작하면 결과를 검토하기 쉽습니다.

2

리믹스

생성된 결과를 그대로 확정하지 말고 목적을 바꿔 다시 요청합니다. 예: 관리자 화면, 모바일 우선, 사내 서비스 톤, 접근성 강화.

3

설정

사용 가능한 AI 에이전트, API 키, 언어, 디자인 시스템을 조직 정책에 맞춥니다. 민감한 프로젝트에서는 허용된 에이전트만 연결합니다.

업데이트와 검증

앱은 아래 메타데이터를 읽어 최신 버전과 플랫폼별 artifact를 확인합니다. 설치 파일을 직접 배포할 때는 같은 릴리스에 포함된 SHA-256 파일로 검증합니다.

  • 최신 메타데이터
  • macOS arm64 manifest
